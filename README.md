# TIPE-RNF

TIPE : Réseaux de neurones formels

Code réalisé dans le cadre de mon TIPE pour le lycée massena pendant l'année 2015/2016

Utilise mon projet YAGE pour la création graphique de poll d'exemples
Utilise la bibliothèque Armadillo pour la gestion des matrices
Utilise boost (python et program option) pour la connexion python-c++ et le petit programme de création des RNF
